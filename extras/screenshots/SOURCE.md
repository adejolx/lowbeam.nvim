# Screenshot sources

The screenshots in this directory are real Neovim captures provided by the theme author.

Mapping used in this archive:

- `lowbeam-day.png` — first light capture
- `lowbeam-dusk.png` — second light capture
- `lowbeam-night.png` — dark capture
- `lowbeam-dark.png` — same dark capture, because `lowbeam-dark` is an alias for `lowbeam-night`

The images were resized to 1600px wide for repository/README use.
